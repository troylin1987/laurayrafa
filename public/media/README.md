# Dónde dejar tus fotos y vídeo

Coloca tus archivos JPEG/MP4 exactamente en estas rutas:

## Hero (foto principal)
- `public/media/novios/hero-principal.jpeg`

## Fotos por separado de los novios
- `public/media/novios/laura.jpeg`
- `public/media/novios/rafa.jpeg`

## Foto de novios jóvenes (inicio timeline 2007)
- `public/media/timeline/2007-primer-encuentro.jpeg`

## Resto de fotos del timeline
- `public/media/timeline/2009-primer-viaje.jpeg`
- `public/media/timeline/2025-modo-cooperativo.jpeg`
- `public/media/timeline/2026-propuesta-epica.jpeg`

## Vídeo opcional de apertura
- `public/media/video/trailer-novios.mp4`
- `public/media/video/poster-video-novios.jpeg`

Si los nombres coinciden con estos, la web los mostrará automáticamente.
